## Git Facts

by Yuzuki Harada

git is a kind of version control software, similar to the "track changes" feature of Microsoft Word or Google Documents.

git tracks changes to files in a special folder called a repo, which is short for repository.

git keeps a log of all the changes you have made to your files. Each change is called a commit. You always provide a short comment describing what happened in the change.
